## 🎯 四种扫描模式（AI智能选择）

让 AI 帮你选择，或自己指定模式：

### 🚀 [A] 极速快扫
- 扫描 **100个** 核心情报源
- 耗时 **2-3分钟**
- 产出 **5-10条** 精华信号
- → 每日日常：快速看看今天有什么新动向

### 🎯 [B] 精准深度
- 扫描 **500个** 精准源 + 关键词匹配
- 耗时 **10-15分钟**
- 产出 **20-50条** 高价值信号
- → 特定方向调研：深挖某个领域的冷门信号

### 🌐 [C] 全网广扫
- 扫描 **126,500+** 全量情报源
- 耗时 **30-60分钟**
- 产出 **100-500条** 含噪声原始信号
- → 每周大盘：地毯式搜索，不放过任何角落

### 🔥 [D] 专项突袭
- 扫描自定义：**关键词 + 指定源 + AI分析**
- 耗时 **5-30分钟**（取决于关键词）
- 产出 **按关键词密度**
- → 追热点 / 监控竞品 / 深度调查

## 💎 五大核心能力

### 1️⃣ 信号降噪引擎
3秒判断信息是"黄金"还是"垃圾"：

| ✅ 高价值信号 | ❌ 丢弃的垃圾 |
|-------------|-------------|
| 零回复但内容极度具体 | 精美排版+流程图 |
| 有操作细节、无引流链接 | "懂的来/私信我" |
| 有数据/截图/源码证据 | 引导未知平台充值 |
| 发帖人有技术历史 | 只有收益截图无过程 |
| 多平台偶然提到同一事 | 同内容多板块发布 |

### 2️⃣ 圈内黑话破译
| 黑话 | 真实含义 |
|:----|:---------|
| 低调撸 | 已实际操作获益 |
| 卡bug | 规则漏洞可利用 |
| API放开/开源了 | 技术门槛骤降 |
| 跑通了 | 流程验证通过 |
| 秒下 | 收益秒到账 |
| 漏了 | 接口/漏洞暴露 |
| 无实名 | 无需身份验证 |
| 免梯 | 无需翻墙 |
| 搬砖 | 跨平台套利 |

### 3️⃣ 逆向工程侦查
```
1. 截图Logo → 识别工具
2. 数据特征 → 推断收益规模
3. whois域名 → 判断项目新鲜度
4. 发帖人历史 → 拼凑技术栈
5. 5+平台交叉 → 验证真实性
6. Archive.org → 还原被删内容
7. Google Cache → 获取已变更页面
```

### 4️⃣ 五维可信度评分 (0-100)
| 维度 | 满分 | 标准 |
|:----|:----:|------|
| 发帖人可信度 | 25 | 技术历史(+15) 老号(+5) 无营销(+5) |
| 信息具体度 | 25 | 有数据(+10) 截图(+8) 步骤(+7) |
| 逻辑自洽度 | 20 | 收益合理(+10) 风险透明(+5) 不完美(+5) |
| 交叉验证度 | 20 | 多平台一致(+10) 可验证(+10) |
| 先手优势 | 10 | 零回复(+5) 无转发(+5) |

**评分：** 80+ 🔥 高价值 | 60-79 👀 重点关注 | 40-59 📝 记录 | <40 🗑️ 噪声

### 5️⃣ MVP 闭环验证
2小时 + 极低成本验证真伪：
```
Step 1: 跑一遍描述流程
Step 2: 有正向收益 → ✅ 逻辑闭环成立
Step 3: 跑不通 → ❌ 伪信号，自动降权
Step 4: 更新可信度评分
```
