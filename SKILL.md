## 🎯 五大核心能力

### 能力一：全网信号降噪 Engine
3秒判断一条信息是"高价值"还是"噪声"：

```
✅ 高价值信号特征：
   1. 零回复/零点赞但内容极度具体 ← 没人注意的宝藏
   2. 有具体操作细节（非纯情绪输出）
   3. 有可验证的客观证据（截图/数据/源码）
   4. 发帖人有技术/实操历史（非营销号）
   5. 多个无关平台偶然提到同一件事 ← 交叉验证

❌ 丢弃的垃圾信号：
   1. 精美排版 + 流程图 ← 引流定式
   2. "懂的来/私信我/加微信" ← 割韭菜卖课
   3. 引导去未知平台充值 ← 资金盘/诈骗
   4. 同内容多板块重复发布 ← 批量引流
   5. 只有收益截图没有操作过程 ← 假信号
```

### 能力三：逆向工程侦查
从一条隐秘帖子反向还原完整链路：
```
1. 提取截图中的极小Logo → 识别工具/平台
2. 分析提及的数据特征 → 推断收益规模
3. 反查域名注册时间（whois）→ 判断项目新鲜度
4. 追溯发帖人历史提问 → 拼凑技术栈
5. 交叉对比5+平台同线索 → 验证真实性
6. 查Archive.org历史版本 → 还原被删内容
7. Google Cache 快照 → 获取已变更页面
```

### 能力四：可信度评分系统
每条情报自动打分（0-100）：

| 维度 | 满分 | 评分标准 |
|:----:|:----:|----------|
| 发帖人可信度 | 25 | 有技术历史(+15)、老号(+5)、无营销(+5) |
| 信息具体度 | 25 | 含数据(+10)、截图/链接(+8)、操作步骤(+7) |
| 逻辑自洽度 | 20 | 收益合理(+10)、风险透明(+5)、不完美(+5) |
| 交叉验证度 | 20 | 多平台一致(+10)、可验证(+10) |
| 先手优势 | 10 | 零回复(+5)、无大V转发(+5) |

**评分解读：**
- **80+** 🔥 高价值信号，值得深入研究
- **60-79** 👀 重点关注，进一步验证
- **40-59** 📝 记录观察
- **<40** 🗑️ 噪声

### 能力五：MVP 闭环验证
2小时 + 极低资金验证信息真伪：
```
Step 1: 亲自跑一遍对方描述的流程
Step 2: 跑通产生正向收益 → ✅ 逻辑闭环成立
Step 3: 跑不通 → ❌ 记录为"伪信号"，下次自动降权
Step 4: 整理验证结果 → 更新可信度评分
```
