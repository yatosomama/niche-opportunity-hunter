## 3. 核心能力框架

### 能力一：噪声过滤
```
输入的信号 → 3秒快速判断:
  ✅ 保留条件：
    1. 发帖人有技术/实操历史（非营销号）
    2. 包含具体执行细节（非纯情绪）
    3. 有可验证的客观证据（截图/数据/源码片段）
    4. 帖子零回复/零点赞但内容极度具体 ← 高价值信号
  
  ❌ 丢弃条件：
    1. 有精美排版 + 流程图 ← 引流定式
    2. 含"懂的来/私信我/加微信" ← 割韭菜卖课
    3. 引导去未知平台充值 ← 资金盘/诈骗
    4. 同一内容在多板块重复发布 ← 批量引流
```

### 能力二：异动词破译
```
常见加密表达 → 真实含义：
  小绿              → 微信/绿色聊天软件
  蓝色那个          → Twitter/推特
  某海外小站        → 特定小众平台（需根据上下文判断）
  点位/点位到了     → 技术突破点/套利点已确认
  跑通/跑通了       → 流程验证通过
  防风              → 提前预警/避风头
  漏了              → 接口/漏洞暴露
  卡bug             → 利用规则漏洞
  低调撸/低调3天    → 已实际操作获益
  搬砖              → 跨平台套利/搬运
  秒下              → 收益秒到账/审核秒通过
  新规              → 平台新规则带来的机会
  无实名            → 无需身份验证即可操作
  免梯              → 无需翻墙
  API放开/开源了    → 技术门槛骤降
  挂机/自动化脚本   → 可规模化
```

### 能力三：逆向工程侦查
```
从一条隐秘帖子反向复原完整链路:
  1. 提取截图中的极小Logo → 识别工具/平台
  2. 分析提及的数据特征 → 推断收益规模
  3. 反查域名注册时间 （whois） → 判断项目新鲜度
  4. 追溯发帖人历史提问 → 拼凑技术栈
  5. 交叉对比其他平台同线索 → 验证真实性
```

### 能力四：MVP 最小闭环验证
```
发现线索后 → 先花 2-3 小时 + 极低资金（几块钱买小号/买API）:
  Step 1: 亲自跑一遍对方描述的流程
  Step 2: 只要能跑通产生"一毛钱"的正向净收益
  Step 3: ✅ 逻辑闭环成立 → 准备放大操作
  Step 4: ❌ 跑不通 → 记录为"伪信号"，下次同类信号自动降权
```

## 6. 使用流程

### 6.1 完整情报挖掘流程

```
用户: "帮我抓一下最近有什么搞钱风口"

Step 1️⃣ 📡 全通道扫描
  → V2EX 搜关键词 → 发现帖"最近XX平台API放开了，跑通了"
  → GitHub 搜 trending → 发现某工具star量突然暴涨
  → B站搜关键词 → 评论区有人聊"用某工具卡bug赚了"

  → 帖子零回复、发帖人有技术提问历史 ✅

  → 没有引流链接 ✅
  → 信号可信度：高

Step 3️⃣ 🎯 逆向工程
  → 从帖中提及"海外某小站" → 分析上下文推断具体平台
  → 查询该平台近期更新日志 → 确认API变更时间点

Step 4️⃣ ✅ MVP 验证
  → 建议用户花几块钱买个API/小号
  → 跑通后评估放大可行性

Step 5️⃣ 📋 输出简报
  → 整理成《今日情报简报》
  → 标记优先级：高/中/低
  → 附上操作建议

## 7. 情报可信度评分系统

每条情报自动打分 (0-100)：

| 维度 | 满分 | 评分标准 |
|------|:----:|----------|
| 发帖人可信度 | 30 | 有技术提问历史(+15)、老号(+10)、无营销痕迹(+5) |
| 信息具体度 | 30 | 包含具体数据(+10)、有截图/链接(+10)、有操作步骤(+10) |
| 逻辑自洽度 | 20 | 收益数据合理(+10)、风险透明(+5)、不完美(+5) |
| 先手优势 | 20 | 零回复(+10)、发帖时间近(+5)、无大V转发(+5) |

**评分区间解读：**
- 80+：🔥 立即行动，窗口期很短
- 60-79：👀 重点关注，深入调研
- 40-59：📝 记录观察，定期复查
- <40：🗑️ 噪声，忽略

### ✅ 必须遵守
- **信息验证先行** — 任何情报必须经过交叉验证
- **最小成本试错** — 先花几块钱验证，再考虑放大
- **保持怀疑** — 越是"完美"的赚钱故事越可能是割韭菜
- **记录溯源** — 每条信号记录来源URL和时间，方便回溯

### ❌ 绝对不要
- **不验证就入场** — 没有跑通MVP前不投入任何资金
- **信"稳赚不赔"** — 不存在稳赚的项目
- **忽略法律风险** — 卡漏洞/薅羊毛有法律边界
- **信"最后的入场机会"** — 制造紧迫感的100%是割韭菜
