## 🎯 五大核心能力

### 1️⃣ 全网信号降噪 Engine
```
✅ 高价值信号特征：
   · 零回复/零点赞但内容极度具体 ← 没人注意的宝藏
   · 有具体操作细节（非纯情绪输出）
   · 有可验证的客观证据（截图/数据/源码）
   · 发帖人有技术/实操历史（非营销号）
   · 多个无关平台偶然提到同一件事 ← 交叉验证

❌ 丢弃的垃圾信号：
   · 精美排版+流程图 ← 引流定式
   · "懂的来/私信我" ← 割韭菜卖课
   · 引导未知平台充值 ← 资金盘/诈骗
   · 只有收益截图无过程 ← 假信号
```

### 2️⃣ 圈内黑话破译器

| 低调撸 | 已操作获益 | 🔥🔥🔥 |

### 3️⃣ 逆向工程侦查
```
1. 截图极小Logo → 识别工具
2. 数据特征 → 推断收益规模
3. whois域名 → 判断新鲜度
4. 发帖人历史 → 拼凑技术栈
5. 5+平台交叉 → 验证真实性
6. Archive.org → 还原被删内容
7. Google Cache → 获取变更页面
```

### 4️⃣ 可信度评分系统 (0-100)
| 维度 | 满分 | 标准 |
|:----|:----:|------|
| 发帖人可信度 | 25 | 技术历史(+15) 老号(+5) 无营销(+5) |
| 信息具体度 | 25 | 数据(+10) 截图(+8) 步骤(+7) |
| 逻辑自洽度 | 20 | 收益合理(+10) 风险透明(+5) 不完美(+5) |
| 交叉验证度 | 20 | 多平台一致(+10) 可验证(+10) |
| 先手优势 | 10 | 零回复(+5) 无转发(+5) |

**80+ 🔥 高价值 | 60-79 👀 重点关注 | 40-59 📝 记录 | <40 🗑️ 噪声**

### 5️⃣ MVP 闭环验证
```

Step 1: 跑一遍描述流程
Step 2: 有正向收益 → ✅ 成立

Step 4: 更新可信度评分
```
